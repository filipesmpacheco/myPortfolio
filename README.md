## 🇧🇷 Sobre este projeto

Este é o meu portfólio pessoal: uma página única, bilíngue (**PT/EN com switch dinâmico**), construída com **HTML, CSS e JavaScript puros** — sem frameworks, sem dependências, sem build. Abra e pronto.

### Destaques

- **Bilíngue de verdade** — troque entre português e inglês com um clique; a escolha fica salva no navegador.
- **Tema claro/escuro automático** — respeita a preferência do seu sistema.
- **Responsivo** — do celular ao ultrawide.
- **Acessível** — HTML semântico, navegação por teclado e atributos ARIA.
- **Currículos inclusos** — PDF e DOCX, em português e inglês, na pasta [`cv/`](cv/).

### Como rodar

```bash
git clone https://github.com/filipesmpacheco/myPersonalPortfolio.git
cd myPersonalPortfolio
open index.html   # ou simplesmente arraste para o navegador
```

Sério, é só isso. Zero `npm install`. 😄

### Estrutura

```
├── index.html            # A página do portfólio
├── assets/
│   ├── css/styles.css    # Design tokens + layout (dark/light)
│   └── js/
│       ├── i18n.js       # Dicionário PT/EN + language switcher
│       └── main.js       # Bootstrap da página
├── cv/                   # Currículos prontos (PDF + DOCX, PT + EN)
└── cv-src/               # Fontes dos currículos (HTML/Markdown)
```

## 🇺🇸 About this project

This is my personal portfolio: a single, bilingual page (**PT/EN with a dynamic switch**) built with **plain HTML, CSS and JavaScript** — no frameworks, no dependencies, no build step. Just open it.
